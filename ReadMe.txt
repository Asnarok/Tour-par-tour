Descritpion:
Il s'agit d'un jeu tour par tour en 1 contre 1 assez classique 
dans lequel les joueurs peuvent choisir un personnage parmi une
liste. Les personnages disposent de différentes attaques,
resistances et faiblesses ainsi que de potions de soin.
A chaque tour, le joueur peut choisir d'utiliser une attaque
contre le personnage adverse ou bien de soigner son propre 
personnage. Le jeu se termine lorsque l'un des deux personnages
n'a plus de point de vie. 

Usage : 
Pour créer un nouveau mode de jeu : 
Créer un fichier texte et repscter le formatage suivant :

[Nom du perso]:
-types: [Type(s) du perso]
-weaknesses: [Faiblesses du perso]
-attacks:
--[Nom de l'attaque]: [Type de l'attaque]
--[Nom de l'attaque]: [Type de l'attaque]

Placer ensuite ce fichier texte ............

Pour jouer : 
Compiler et executer la classe Game puis suivre les instructions du terminal.
